# Bootstrap Inventory Checklist

## Backend inventory
- current app entrypoint
- auth implementation
- DB session setup
- SQLAlchemy model base
- existing tables
- Alembic config and latest revision
- current tests
- current environment loading behavior

## Frontend inventory
- UI framework
- routing structure
- nav structure
- theme system
- current dashboard/page model
- typed API client status

## Deployment inventory
- local runtime version
- CI runtime version
- production runtime version
- environment variable source
- database URL source
- WSGI/ASGI entrypoint
