# hospitable-web (skeleton)

Next.js skeleton for the Hospitable Web frontend. Run `npm install` then `npm run dev` to start after adding NEXT_PUBLIC_OPS_API_BASE_URL.

Quickstart

1. Install dependencies:
   - cd hospitable-web
   - npm install

2. Start dev server:
   - NEXT_PUBLIC_OPS_API_BASE_URL=http://localhost:8000 npm run dev

3. Build for production:
   - npm run build && npm run start

Notes
- Components are TypeScript-ready; install types if needed.
- See HN3T_MASTER_PLAN.md for architecture and API contract guidance.
